Que es HTML

es un lenguaje de marcado esencial para el desarrollo

Etiquetas

Cada elemento tiene una etiqueta asociada. Podemos crear un
elemento usando su etiqueta en nuestro arquivo HTML
Estos sonejemplos de etiquetas usadas comun mente

  • < html > -todo el contenido de la páguina wdebe estar entre esta
    etiqueta

  • < head >-este elementocontiene el titulo de la páguina que
    puedes ver en la pestaña de tu navegador y además contiene
    metadatos

  • < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 > - estos elementos
    representan encabezados o títulos de mayor a menor
    importancia o nivel en la página web.

  • < p >- un párafo

  • < a >- un enlace a otra página web o a secciones internas de tu
    página web.

    • < strong > - esta etiqueta te permite resaltar texto importante.
    Usualmente se muestra en negrita.
  • < em > - esta etiqueta te permite resaltar texto importante.
    Usualmente se muestra en cursiva.

  • < input > - un elemento que te permite crear formularios
    porque puede tomar texto o actuar como un botón de radio o
    una casilla de verificación.

  • < br > - una línea horizontal que puede ser usada para separar
    párrafos o secciones de tu página web.

  • < footer > - un pie de página.

La mayoría de los elementos en HTML tienen una etiqueta de
apertura y una etiqueta de cierre. Entre ambas etiquetas se escribe el
contenido del elemento.

Comentarios

HTML también nos permite escribir comentarios. Los comentarios
son muy útiles para agregar notas que nosotros y otros
desarrolladores pueden leer al ver el archivo. Son muy útiles para
explicar y entender la estructura de la página y saber qué tareas están
pendientes:

Dato: Los comentarios no son parte del código ni del resultado
final. Son útiles para los desarrolladores que trabajan con el archivo
HTML.

Liatas

En HTML puedes crear listas ordenadas y listas no ordenadas con las
etiquetas < ol > y < ul >, respectivamente.

Aquí tenemos un ejemplo de cada tipo de lista:

  1. Azul
  2. Verde

  • Azul
  • Verde
  • Negro

Tip: La diferencia entre ambos tipos de listas es que los elementos
de las listas ordenadas se muestran con una numeración específica
mientras que los elementos de las listas no ordenadas muestran
viñetas sin numeración.
